Welcome

This repo currently hosts static HTML assets. A new React-ready radial orbit component has been added under /components/ui so that the project can graduate toward a shadcn + Tailwind + TypeScript frontend stack. Use this page as a quick start for wiring everything up.

Component drop point

The default shadcn convention is /components/ui for shared primitives; the directory now exists with radial-intro.tsx and a demo scaffold (demo.tsx). Keeping UI primitives in this folder matters because the shadcn CLI writes new components there by default, making imports consistent and discoverable across the app.

Styles are expected to live in a global stylesheet such as /app/globals.css (Next.js) or src/index.css (Vite/CRA). Tailwind layers should be registered there once the build tooling is in place.

Radial intro component

The component animates orbiting avatars. It depends on motion/react and React 18+. Pass an orbitItems array with id, name, and src fields. Optional props stageSize and imageSize control layout.

  1. Ensure dependencies are installed: npm install motion react react-dom plus TypeScript types (@types/react, @types/react-dom).
  2. Add an alias for @/components in tsconfig.json or jsconfig.json: "paths": { "@/*": ["./*"] } so imports like "@/components/ui/radial-intro" resolve.
  3. Render the demo with <DemoOne /> from /components/ui/demo.tsx; it already points at nine Unsplash images with permissive hotlinking.

shadcn + Tailwind + TypeScript bootstrap

1) Add TypeScript + React

Install React 18, TypeScript, and typings: npm install --save react react-dom && npm install --save-dev typescript @types/react @types/react-dom. Run npx tsc --init and set "jsx": "react-jsx" with "baseUrl": "." for path aliases.

2) Tailwind CSS

npm install -D tailwindcss postcss autoprefixer, then npx tailwindcss init -p. Point content to ./{pages,components,app}/**/* and add the three Tailwind directives to your global CSS file.

3) shadcn CLI

Run npx shadcn-ui@latest init and choose components as the output folder (it will target /components/ui). Generate primitives with npx shadcn-ui@latest add button card avatar as needed.

4) Wire the demo

Import DemoOne into your landing page (e.g. app/welcome/page.tsx or src/App.tsx) and wrap the app with Tailwind's base styles. Adjust stageSize or imageSize props for different layouts.

Notes & open tasks

  • Install motion: add it to package.json. If npm registry access is limited, retry installation when connectivity is available so the lockfile stays in sync.
  • Icons: prefer lucide-react for any new SVGs to stay consistent with shadcn defaults.
  • Responsive behavior: the orbit scales with stageSize; consider wrapping it in a flex container to center on mobile screens.
